Примеры

Ниже приведены примеры использования EDOC Schema. Пример документа - заявление на оказание муниципальной услуги по 210-ФЗ

Простые

Пример ИНН

Схема:

{
    "id": "http://edoc-schema.kzn.ru/identifier/ru/inn",
    "$schema": "http://json-schema.org/draft-06/schema#",
    "description": "INN",
    "type": "string",
    "pattern": "^(?:\\d{10}|\\d{12})$"
}

Данные:

"160123456789"

Пример ОКОПФ

Схема:

{
    "id": "http://edoc-schema.kzn.ru/identifier/ru/ogrn",
    "$schema": "http://json-schema.org/draft-06/schema#",
    "description": "OGRN",
    "type": "object",
    "properties": {
      "code": {"type": "number"},
      "title": {"type": "string"}
    }
}

Данные:

{
  "code": "12300",
  "title": "Общество с ограниченной ответственностью"
}

ФИО

Схема:

{
    "id": "http://edoc-schema.kzn.ru/person/fio",
    "$schema": "http://json-schema.org/draft-06/schema#",
    "description": "Surname, name and patronic of a person",
    "type": "object",
    "properties": {
        "surname": { "type": "string" },
        "name": { "type": "string" },
        "patronimic": { "type": "string" }
    }
}

Данные:

{
  "surname": "Пушкин",
  "name": "Александр",
  "patronimic": "Сергеевич"
}

Составные

ФИО и номер телефона

Схема:

{
    "id": "http://edoc-schema.kzn.ru/person/fio_phone",
    "$schema": "http://json-schema.org/draft-06/schema#",
    "description": "Surname, name, patronic of a person, and his phone",
    "type": "object",
    "properties": {
      "fio": {"$ref": "http://edoc-schema.kzn.ru/person/fio"},
      "phone": {"$ref": "http://edoc-schema.kzn.ru/generic/phone_number"}
    }
}

Данные:

{
  "fio": {
    "surname": "Лобачевский",
    "name": "Николай",
    "patronimic": "Иванович"
  },
  "phone": {
    "country_code": 7,
    "region_code": 843,
    "number": 2599923
  }
}

Юридическое лицо

Схема:

{
    "id": "http://edoc-schema.kzn.ru/organization/ru/organization",
    "$schema": "http://json-schema.org/draft-06/schema#",
    "description": "Russian organization properties",
    "type": "object",
    "properties": {
        "inn": { "$ref": "http://edoc-schema.kzn.ru/identifier/ru/inn" },
        "ogrn": { "$ref": "http://edoc-schema.kzn.ru/identifier/ru/ogrn" },
        "kpp": { "$ref": "http://edoc-schema.kzn.ru/classifier/ru/kpp" },
        "okopf": { "$ref": "http://edoc-schema.kzn.ru/classifier/ru/okopf" },
        "name": { "$ref": "http://edoc-schema.kzn.ru/generic/name" }
    }
}

Данные:

{
  "inn": "1657119572",
  "ogrn": "1121690046411",
  "kpp": "165701001",
  "okopf": "12300",
  "name": {
    "full": "Рога и копыта",
    "short": "Рога и копыта",
    "abbreviation": "РиК",
    "common": "Рога"
  }
}
